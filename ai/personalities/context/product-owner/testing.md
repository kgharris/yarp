# Product Owner — Testing Phase

## Focus

- Are UX test scenarios grounded in real user tasks and workflows?
- Are there workflows promised in requirements that have no test coverage at all?
- Does the test suite verify the tool's core value proposition end-to-end — can the user answer "am I on track?" and trust the answer?

## Artifacts

Review UX/integration test scenarios against requirements for coverage gaps. Do not assess test mechanics — that's for the QA engineers.
